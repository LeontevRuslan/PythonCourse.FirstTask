# TaskManager

Проект **TaskManager** представляет собой веб-приложение для управления проектами и задачами. Он позволяет пользователям создавать, редактировать, удалять проекты и задачи, а также отслеживать их статус. Этот проект использует **Django** как фреймворк для создания веб-приложений и **PostgreSQL** как базу данных.

## Описание проекта

Это простое веб-приложение позволяет пользователям:
- Создавать проекты и задачи, которые к ним привязаны, с описанием, сроком выполнения и статусов.
- Отслеживать состояние задач.
- Редактировать или удалять задачи.
- Использовать фильтрацию и сортировку задач и проектов по различным критериям.

В проекте используются следующие технологии:
- **Django** — веб-фреймворк для быстрого создания веб-приложений.
- **PostgreSQL** — реляционная база данных для хранения данных о задачах.
